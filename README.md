# 6th Template - Travel Landing Page

A modern, animated travel website template built with Next.js, TypeScript, Tailwind CSS, shadcn/ui conventions, and Framer Motion.

## Tech Stack

- Next.js 13
- TypeScript
- Tailwind CSS
- shadcn/ui structure
- Framer Motion
- Lucide React

## Key Features

- Fullscreen hero with animated background image slider
- Premium expanding travel navbar with smooth transitions
- Marquee + animated stats strip
- Tailored experiences section (inspiration-based card layout)
- Interactive expanding image gallery
- Popular destinations grid with ratings and pricing
- 3D vertical image stack (drag/scroll interaction)
- Step-by-step process section
- Animated testimonials carousel
- Parallax-style CTA banner
- Modern contact form and footer

## Project Structure

- `app/page.tsx` - Main page composition
- `components/ui/*` - Reusable UI components (navbar, gallery, image stack)
- `components/landing/*` - Modular landing sections

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` (or the next available port shown in terminal).

## Notes

- This template follows the shadcn-style component organization (`components/ui`).
- External images are used from Unsplash.
- Animations are optimized for a smooth modern landing-page experience.

## Conclusion

This template is designed as a high-conversion travel landing page with strong visual storytelling, motion-driven interactions, and modular section architecture, making it easy to extend with real content, API integrations, and booking workflows.
